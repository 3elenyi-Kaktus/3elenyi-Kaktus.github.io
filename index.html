Данный визуализатор представляет собой один из вариантов реализации структуры данных "Персистентная очередь". Основная концепция алгоритма - построение на двух деревьях. Одно из них представляет саму очередь, второе является вспомогательным для совершения различных операций над ней.

Инструкция по интерфейсу

Перед началом работы настоятельно советуется прочитать техническое описание алгоритма, для понимания его составных частей.

1. Меню

Здесь можно изменить язык, просмотреть различные конфигурации и тд.

1.1 Перестроить граф

Если в процессе работы граф потерял структурированность (вершины перекрывают друг друга, надписи смешиваются), то эта опция повторно запустит перестройку графа.

2. Основной интерфейс

2.1 В верхней части расположена строка с объектами Версий. При нажатии на Версию будут выделены соответствующая ей очередь, а также ее вспомогательные списки.

2.2 В левой части расположен основной граф - сама очередь.

2.3 Справа расположено дерево вспомогательныых списков.

2.4 Чтобы добавить или удалить вершину из определенной версии очереди, введите номер искомой версии в соответствующее поле и нажмите кнопку.

2.5 Если вы хотите, чтобы весь процесс добавления/удаления происходил пошагово и наглядно - отметьте поле Пошаговая визуализация.
